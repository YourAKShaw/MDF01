# MDF01
A RESTful API for a simple e-commerce application using Node.js and MySQL. This API should manage products, categories, and user authentication.

### Running The Project

- Build the Project:

  ```
  npm run build
  ```

- Start the Project:

  ```
  npm start
  ```

- Develop the Project with Auto-Rebuild and Restart:

  ```
  npm run dev:build
  ```
